üéØ Project Overview

Build a feedforward neural network from scratch to predict credit risk (loan defaults) using tabular financial data. The model will provide uncertainty estimates using Monte Carlo Dropout to determine when predictions are reliable vs when human review is needed.

üìö Key Learning Objectives

  • Build feedforward networks for tabular data (no CNNs)
  • Handle real-world data: missing values, scaling, encoding
  • Implement uncertainty quantification with MC Dropout
  • Address class imbalance in financial data
  • Evaluate with financial metrics (precision/recall, ROC-AUC)
  • Deploy model decisions with risk thresholds

üî¨ Project Components

Phase 1: Data Processing & EDA

Load credit data (Lending Club, German Credit, or similar), handle missing values, feature engineering

Phase 2: Feedforward Network Implementation

Build multi-layer network from scratch with ReLU, dropout, batch normalization

Phase 3: Training with Class Imbalance

Implement weighted loss, SMOTE, or focal loss to handle imbalanced defaults

Phase 4: Uncertainty Quantification

MC Dropout for confidence estimation, calibration curves, decision thresholds
